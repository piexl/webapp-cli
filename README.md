# webapp-cli
一个为构建响应式webapp的构建工具。构建响应网站模板[https://github.com/piexl/tplPc.git](https://github.com/piexl/tplPc.git)

###安装插件
```python
npm install pixel-webapp-cli
```

###初始化项目

```python
webapp-cli -i <projectname>
```
####MAC bug修复方案
Mac上安装后会出现错误
```python
env: node\r: No such file or directory with cordova cli
```
